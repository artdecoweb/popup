## API

The package is available by importing its default function:

```js
import popup from '@lemuria/popup'
```

%~%

```## popup => Window
[
  ["url", "string"],
  ["title", "string"],
  ["width", "?number"],
  ["height", "?number"],
  ["features", "Object="],
]
```

Opens a popup in the middle of the screen, and returns the reference to the window.
